# 如何通过GitHub网站访问您的代码

## 访问步骤

### 1. 打开GitHub网站

在浏览器中输入以下网址打开GitHub官方网站：
```
https://github.com
```

### 2. 登录您的GitHub账号

如果您已经登录，请跳过此步骤。如果未登录：
- 点击页面右上角的"Sign in"按钮
- 输入您的GitHub用户名/邮箱和密码
- 点击"Sign in"完成登录

### 3. 访问您的代码仓库

您可以通过以下两种方式之一访问您的代码仓库：

#### 方式一：直接访问仓库URL

在浏览器地址栏中输入以下URL：
```
https://github.com/gjj-mjm/GJJ-MJM.github.io
```

#### 方式二：通过个人主页访问

1. 登录后，点击页面右上角的个人头像
2. 从下拉菜单中选择"Your repositories"
3. 在仓库列表中找到并点击"GJJ-MJM.github.io"

## 浏览和管理代码

### 查看文件结构

进入仓库后，您可以看到项目的文件和文件夹结构。点击任何文件夹可以进入并查看其内容。

### 查看文件内容

点击任何文件可以在线查看其内容。对于代码文件，GitHub会提供语法高亮显示。

### 下载代码

如果您需要将代码下载到本地：

1. 在仓库页面点击绿色的"Code"按钮
2. 选择"Download ZIP"选项
3. 下载完成后，解压ZIP文件即可获得完整代码

### 克隆代码（推荐开发者使用）

如果您熟悉Git命令，可以使用以下命令克隆仓库：

```bash
git clone git@github.com:gjj-mjm/GJJ-MJM.github.io.git
```

## 查看提交历史

在仓库页面点击"Commits"选项卡，可以查看所有提交历史，包括我们刚才推送的提交记录。

## 其他操作

### 查看分支

点击"master"下拉菜单可以查看仓库中的所有分支。目前我们只使用了master分支。

### 设置仓库

点击"Settings"选项卡可以修改仓库的各种设置，如仓库名称、描述、协作者权限等。

## 注意事项

- 确保您使用的是正确的GitHub账号，该账号有权限访问gjj-mjm/GJJ-MJM.github.io仓库
- 如果遇到访问权限问题，请检查您的账号是否与仓库关联
- 您可以在仓库页面的README中添加项目说明文档，使项目更加完善
